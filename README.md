# Android_Firebase
Baby android app that (hopefully) does Firebase login

## Android version
- compileSdk: 31
- minSdk: 31
- targetSdk: 31

Runs on Pixel 5a (Android 12)

## Notes
- Don't save password on phone
- Enable SHA1 fingerprint in firebase console
